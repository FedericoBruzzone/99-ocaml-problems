# **Learn OCaml**

# How to compile

```Bash
$ make all                        # Compile and run all files
$ make clean
```

# Set up

```Bash
$ opam init          # Can take some time
$ eval $(opam env)
```

or

```Bash
$ opam switch create 4.14.0
$ eval $(opam env)
```
